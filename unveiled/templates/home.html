Deep learning experiments.


The purpose of this project is to provide an exploration tool for recent deep learning techniques.

Right now, the application currently supports:

  • Face detection, based on face_recognition python package (dlib).

Tutorial: please use the data selection widget on the left panel to explore the algorithm's capabilities.

Limitations: currently, detection accuracy is not very precise on face detection for unknown images. The model may need to be retrained and more robust for vector simple transforms (faces rotated/translated).


Links: You can access the github of the project at greyzor/unveiled-deep-app. Please feel free to contribute, feedbacks are welcome.
